Ben 10 Alien Gallery Website

A fan-made Ben 10 website built using HTML, CSS, Bootstrap, and JavaScript.

Features

- Home page with Ben 10 theme
- Alien selection page
- Individual pages/sections for:
  - Diamondhead 💎
  - XLR8 ⚡
  - Cannonbolt 🟡
- Image carousel for alien galleries
- Custom alien-themed gradient backgrounds
- Responsive layout using Bootstrap
- Interactive navigation between sections/pages

Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- JavaScript

Project Structure

index.html
index.css
images/

Aliens Included

Diamondhead

- Species: Petrosapien
- Home Planet: Petropia
- Powers: Crystal weapon creation, shields, and durability

XLR8

- Species: Kineceleran
- Home Planet: Kinet
- Powers: Super speed, agility, and reflexes

Cannonbolt

- Species: Arburian Pelarota
- Home Planet: Arburia
- Powers: Rolling attacks, armor, and impact resistance

This project was created to practice front-end web development concepts including:

- Page layouts
- Navigation
- CSS styling
- Bootstrap components
- JavaScript interactions
